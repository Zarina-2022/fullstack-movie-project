 ## Routes
```bash
    GET    /api/movies         > get all movies
    GET    /api/movies/:id    > get movie by id
```
```bash
    POST   /api/movies         > add a new film
    DELETE /api/movies/:id    > delete movie by id
```
 ## To run the server:

 Type 'npm run nodemon'(enter) command in the terminal.
